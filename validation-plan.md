# Gadugy — Validation Plan v1

## Objective

Validate whether values-driven LA homeschool/pre-homeschool families want a curated, private local matching platform strongly enough to complete a detailed intake, accept verification, connect with matches, and eventually pay.

## Phase 1 — Brand + Landing Page Test

### Hypothesis
Families respond to a warm, rooted, wholesome brand promise more than a political/conservative frame.

### Test
Use landing page with headline:
> Meet local homeschool families who share your hopes for your children’s future.

### Measure
- Visit-to-email conversion
- Email-to-intake completion
- Qualitative comments
- Which language resonates: values-driven, homeschool, local village, curated, parent-only, faith-friendly

## Phase 2 — Intake Completion Test

### Hypothesis
The right families will complete a detailed intake if it clearly improves match quality and trust.

### Measure
- Start-to-complete rate
- Drop-off point
- Time to complete
- Which questions feel too sensitive
- Which questions families appreciate

### Special Questions to Watch
- Faith/worldview
- Technology attitudes
- AI matching preference
- Dealbreakers
- ID verification comfort
- Willingness to pay

## Phase 3 — Matching Mode Test

### Hypothesis
Families want curated recommendations first, with some ability to browse.

### Variants
1. Strict recommended matches only
2. Balanced recommended + limited browse
3. Open browsing with filters

### Measure
- Match request rate
- Mutual acceptance rate
- Messages started
- Parent-reported trust/comfort
- Actual meetup intent or completion

## Phase 4 — Verification Timing Test

### Hypothesis
ID verification increases trust but may reduce conversion if asked too early.

### Variants
1. ID before profile approval
2. ID before appearing in matches
3. ID before messaging
4. ID before events only

### Measure
- Completion rate
- Trust rating
- Drop-off rate
- Support questions/concerns
- Parent comments

### Keep this copy
> To keep this community safe and parent-only, every family is reviewed before joining. ID verification helps us confirm that real local parents are behind each profile. Your ID is never shown to other families.

## Phase 5 — Willingness to Pay Test

Do not charge before basic liquidity unless testing a founding cohort.

### Models to test
1. Free early access
2. $49/year founding family membership
3. $9–$19/month membership
4. Free to join, paid to message/connect
5. Paid curated local events

### Recommended order
1. Free early access until enough local families exist
2. Offer optional $49/year founding family membership once value is credible
3. Test $9–$19/month after match quality is proven
4. Add paid events/circles once local clusters form

## Success Criteria for Build Investment

Strong signal if:
- 100+ LA families join waitlist
- 40%+ complete detailed intake
- 50%+ say ID verification increases trust or is acceptable
- 30%+ request at least one match/intro
- 20%+ receive mutual match
- 10+ families express willingness to pay or join founding cohort

## Kill/Park Criteria

Park if:
- Families like the idea but refuse detailed intake
- LA family acquisition is too slow without paid ads
- ID verification kills trust/conversion
- Matching criteria are too fragmented to create local liquidity

Kill if:
- Families only want free Facebook-style community
- No one wants introductions after intake
- Values alignment is too sensitive to operationalize safely

## Next Action

Choose top 3 brand candidates, run registrar/domain/trademark/social-handle screening, then build a simple landing page + intake form.
